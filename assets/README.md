# Assets

This folder is the organized home for site assets going forward.

- `css/site.css` is the public stylesheet entrypoint used by the HTML pages.
- Root-level CSS files are kept for backward compatibility and are imported by `site.css`.
- Root-level page screenshots and logo images are still kept at the repository root so existing GitHub Pages links do not break. They can be moved into `assets/images/` later in a dedicated binary cleanup pass.
